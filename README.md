Tato práce se zaměřuje na automatizovanou instalaci a konfiguraci Zabbix Serveru 7.0 LTS a Zabbix Agent2 pomocí nástroje Vagrant a shell skriptů. V rámci projektu je vytvořen plně funkční monitorovací systém včetně webového rozhraní, databáze a propojení agenta se serverem. Součástí práce je také ověření funkčnosti monitoringu, včetně sledování SSL certifikátu školního webu a snímky obrazovek dokazující správný běh celého řešení.
